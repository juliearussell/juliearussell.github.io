Lists Usage
Unordered List An unordered list is used to display a list of items each preceded with a bullet,
where the items do not need to be shown in a specific order. This list uses the <li> tag.
Ordered List An ordered list is used to display a list of items each proceeded with a number,
where each item typically needs to be shown in a specific order,
or where it is useful to know the quantity of items in the list. This list uses the <ol> and </ol> tags.
Definition List A definition list displays items in pairs, like a dictionary, glossary,
or questions and answers. This list uses the <dl> and </dl> tags.

Acronyms, de-acronymized in a definition list

HTML
Hyper Text Markup Language; a non-proprietary language interpreted (not compiled!) by the web browser.
CSS
Cascading Style Sheet; it's not a programming language, but a way to style and format web pages.
FTP
File Transfer Protocol; an Internet protocol and service used to transfer files from one system to another.
SSH
Secure Shell; an Internet protocol and service used to connect to a system over an encrypted connection.
