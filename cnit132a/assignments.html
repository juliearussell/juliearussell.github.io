Homework Assignments
for City College of SF Course
CNIT132A Advanced HTML & CSS

Homework 3: Using images in HTML
The assignment in which we learned to optimize images for enhanced website performance without sacrificing visual quality.

Homework 4 Part 1: Using CSS Grid to layout web pages
The assignment in which we expanded our knowledge of CSS Grid Layout.

Homework 4 Part 1: Using CSS Flexbox to layout web pages
The assignment in which we expanded our knowledge of CSS Flexbox Layout.

Homework 4 Part 1: Using columns to format text on web pages
The assignment in which we learned to layout webpages in columns.

Homework 5: Using Microdata & WAI ARIA Role Attributes to improve accessibility in web pages
Developing webpages that are accessible to everyone is critical and in the USA, it's the law. In this assignment we learned how to use microdata and WAI ARIA role attributes to improve accessibility of our pages.

Homework 6: Using SASS and SCSS to add functions and variables to web pages
Although this assignment was not uploaded as a webpage to our course sites, we learned how to use variables and functions in our CSS code with preprocessing languages Synactically Awesome Stylesheets(SASS) and Sassy CSS (SCSS).

Homework 7 Part 1: Using CSS to add animations to web pages
Animations add visual interest and intrigue to web pages, although it's important to be mindful of visitors who may be sensitive to this kind of movement! In this assignment, we learned to use CSS to animate the cloud background image to make the plane appear to move, and also add a moving banner that respects the dimension of the viewable screen.

Homework 7 Part 2: Wrapping text around images with CSS
This was a fun assignment, where we learned to use CSS Shapes to wrap text around this house shape, and a container query to adapt the size of heading two based on the size of the container.

And so, we arrive at the final two assignments for the Advanced HTML & CSS course!

Homework 8 Part 1: Using Search Words to improve SEO
Without SEO, how will visitors find my website? In this assignment we use keywords, and other elements to improve native SEO.

Homework 8 Part 2: Analytics Report
At the start of this course, I added a google analytics tag to the main four pages in the left hand nav. This is where I show the results and see if anyone visited in the past few months!

If you want to learn HTML and CSS in a collaborative, structured environment, check out the CNIT132 and CNIT132A classes at City College of San Francisco.
