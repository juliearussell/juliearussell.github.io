Helpful Tips for Better Search Engine Optimization (SEO)

Everyone wants their website to show up at the top of a Google search and employing SEO suggestions can make that more likely.

Here are a few ways to help search engines help you:

  • Domain Name:
    Choose a domain name that is representative and easy to distinguish the purpose of one's company, organization, or website, juliesdogcare.com is better for Julie's Dog Care than randomwords.com.
  • If the website will be referenced on social media, which is a good idea to increase site visitors, use Open Graph meta tags so that social media links use specific images. Find out more on The Open Graph Protocol
  • Don't even think about publishing your site without a meta description or title! Google uses this information for their Search Engine Response Tag (SERP), which is Google's response to a user's search query.
  • Using relative links for pages on the same webserver is useful, especially when the dev or staging environment has a different domain name than production, but make sure to code the canonical link: <link rel="canonical" href="https://mycompany.com/"> so that googlebot and other webcrawlers do not duplicate the URL
  • Errors are the enemy of SEO! Validate html at W3.org to find and resolve site errors.

There are many resources and articles available to help optimize SEO, here are a few:

  • Semrush Blog DIY SEO: Six Steps That Anyone Can Follow To Get Results
    This blog post recommends first setting up search analytics, like Google Analytics, to help determine how visible one's site is now. After that, search keywords visitors might use to find the site, and publish optimized content, including the most relevant keyword in the H1 heading.
  • Forbes has a recent article with a list of steps on improving one's SEO campaign in 2022
    The first tip is to optimize the website for page load speed, as site responsiveness plays a role on how long a visitor will stay on one's site. Subsequent tips mirror those provided by Semrush: track your traffic (which can be done with Google Analytics), use accurate keywords, and more.
  • Screaming Frog is a well known SEO Optimization tool, despite its curious name. The SEO Spider Tool is a website crawler that helps improve website SEO by auditing for common SEO issues including broken links, auditing redirects, analyse (they're British) page titles and meta data, generate site maps, and more.
