About me: When I am not swimming with dolphins in Mexico, I work in tech, walk my dog, write fiction, practice yoga, and enjoy time with my friends.

My path into and around web development has been circuitous. I have worked in web infrastructure since 1998, along side designers, front end developers, applications developers, and the like. I'd never coded a web page until 1999 when I spun up a very amateurishly designed web magazine using MS Front Page. It was completely static html. A few years later I created a marketing web page for a yoga business I started that never got off the ground. This design was a little better. You can see that site on the wayback machine. In years since, I have continued working in web infrastructure, or giving the web code a place to call home, but haven't touched html until CS131H this semester. I had forgotten how fun it is to create web pages, and the joy of seeing the pages go live on the Internet. My favorite website is also my last employer, Medium. I've always appreciated the clean design, the user generated content, and my team and I worked hard to keep the infrastructure operational behind the scenes.
